About Our Project

This website serves as a basic example of how to build a multi-page web application using Python's Flask framework. Flask is a lightweight and flexible micro-web framework, perfect for getting started with web development.

We've used Tailwind CSS for styling to ensure a modern, responsive, and aesthetically pleasing design with minimal effort. All elements are designed to adapt beautifully across different screen sizes, from mobile phones to large desktops.

Our goal is to demonstrate a clear and simple structure for a web project, including routing, templating, and basic navigation.
